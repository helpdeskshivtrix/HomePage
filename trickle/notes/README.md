# Shivatrix - Building the Digital Backbone of Tomorrow

## Overview
A futuristic IT services website for Shivatrix, featuring cutting-edge design with animated computer graphics, dark navy theme, and modern technology solutions.

## Structure
- **index.html** - Main homepage with animated hero section
- **app.js** - Main React application with error boundary
- **components/** - React components for each section:
  - Header.js - Navigation with Shivatrix branding
  - Hero.js - Animated hero section with computer SVG graphics
  - Services.js - Service cards with flip animations
  - Footer.js - Site footer with company information
  - ServicesPage.js - Detailed services page
  - AboutPage.js - Company information and values
  - ContactPage.js - Contact form with validation
  - PortfolioPage.js - Project portfolio with filtering

## Design System
- **Primary Color**: #0a1931 (Dark Navy)
- **Secondary Color**: #b3cfe5 (Sky Blue) 
- **Accent Color**: #4a7fa7 (Steel Blue)
- **Tertiary Color**: #1a3d63 (Deep Blue)
- **Light Background**: #f6fafd (Off-White)
